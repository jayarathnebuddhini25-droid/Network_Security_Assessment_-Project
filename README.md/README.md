Network Security Assessment
Project Overview

This project demonstrates a structured Network Security Assessment process using an IT audit and cybersecurity approach. The assessment focuses on identifying network exposure, reviewing available services, evaluating security risks, and recommending improvements to strengthen network security controls.

The project simulates how an IT auditor reviews network security posture through scanning, vulnerability identification, risk assessment, and reporting.

Objectives

The main objectives of this project are:

Identify active network services and potential exposure.
Review open ports and service configurations.
Identify security weaknesses.
Evaluate risks based on likelihood and impact.
Provide recommendations to improve network security.
Scope

The assessment covers:

Network discovery
Port scanning
Service identification
Vulnerability assessment
Risk evaluation
Security recommendations

Key Findings

The assessment identified several potential security concerns:

Exposed remote access services
Unnecessary open ports
Service configuration weaknesses
Limited network monitoring practices
Risk Management Approach

Risks were evaluated using:

Risk Score = Likelihood × Impact

Risk levels were categorized as:

Low
Medium
High
Recommendations

Recommended improvements include:

Review and restrict exposed network services.
Apply secure configuration standards.
Implement stronger access controls.
Improve network monitoring and logging.
Perform regular security assessments.
